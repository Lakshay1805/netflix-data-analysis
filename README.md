# 📊 Netflix Data Analysis

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the Netflix dataset to understand trends in content type, ratings, genres, and country-wise distribution.

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## 🔍 Analysis Performed

* Movies vs TV Shows trend over time
* Country-wise content distribution
* Content rating distribution
* Rating comparison by content type
* Genre (category) analysis

---

## 📈 Key Insights

* Netflix content increased rapidly after 2015, showing major platform expansion.
* Movies dominate the platform, but TV Shows have grown significantly over time.
* The United States contributes the most content, followed by India and the United Kingdom.
* Most content is targeted toward mature audiences (TV-MA, TV-14).
* International and drama genres are the most common.

---

## 📂 Dataset

The dataset is included in this repository as `netflix_titles.csv`.

---

## 🚀 Conclusion

Netflix has expanded rapidly with a strong focus on movies and increasing investment in TV Shows. The platform serves a global audience with diverse content while relying heavily on key regions and popular genres.
